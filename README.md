
## Koa2 + sequelize实践开发接口

使用Koa2+MySQL的一个玩具项目，用于练手。前端view层不定。

```
	"start": "node bin/www",
	"dev": "./node_modules/.bin/nodemon bin/www"
```

运行

```
yarn dev

localhost:3000
```
